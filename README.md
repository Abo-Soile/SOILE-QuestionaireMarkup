# SOILE-QuestionaireMarkup
Questionaire Markup Language for the Soile Web service

# Installation
```
git clone git@github.com:Abo-Soile/SOILE-QuestionaireMarkup.git
cd SOILE-QuestionaireMarkup
mvn install
```
# Create Documentation

```
mvn site
```
